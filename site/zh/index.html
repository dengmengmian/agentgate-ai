AgentGate 是面向 AI 应用和客户端的本地 AI 网关，支持 Codex、Claude Code、Gemini CLI、OpenCode、AtomCode，也可接入兼容 OpenAI、Anthropic、Gemini 协议的应用。它把每一次请求路由到你选择的模型，在供应商故障时自动切换，本地完整追踪每一次请求，并打通不同协议与任意 OpenAI 兼容供应商之间的差异。

$ AI 应用 → Codex · Claude Code · Gemini CLI · OpenCode · AtomCode
$ 上游 → OpenAI · Anthropic · DeepSeek · Kimi · GLM · Copilot 再 + 20 家

# 官方体验不变

保留 Codex / Claude Code / OpenCode / Gemini CLI / AtomCode 等客户端的原有使用方式。Codex Desktop 的插件、Claude Code 的设置、Copilot 的额度都不会因此乱,支持一键恢复官方配置。

// Original client experience intact. One-click restore.

# 模型路由归你管

AI 应用的模型请求进 AgentGate,本地决定转换协议、直连上游、选择 Provider 和模型。26 家 provider,多 key 轮询,故障自动切换 —— 全程跑在你自己的机器上。

// Official API enters AgentGate. You decide — locally.

# 每次请求看得见

每次请求的路由、转换、错误、Token、成本和故障转移都在本地面板可追踪。哪条请求花了多少、跑到哪、为什么转移,都有据可查。

// Routing, conversion, errors, tokens, cost, failover — all auditable.

AgentGate 不是托管 API 分发平台,而是 AI 模型请求的本地桌面入口。 要 provider 自由,又不愿客户端原有能力被破坏。

// Not a hosted API reseller — a local-first desktop workbench.
