# agent-native compatibility

Bridges Codex Responses, Claude Messages, Gemini, and Chat Completions so coding agents can talk to more providers without hand-mapping protocols.

桥接四种协议；让 agent 跨 provider 通用，不用自己改协议。

# one-click client setup

Apply or restore Codex / Claude Code / OpenCode / Gemini CLI / AtomCode configs without hand-editing local files. Switch back to official whenever you want.

一键应用 / 还原客户端配置，不用手改 ~/.codex/config.toml 或 ~/.claude/settings.json。

# cost-aware routing

Routes across 26 providers with failover, capability checks, latency / price strategies, and per-request cost tracking — all on your machine.

26 个 provider 智能路由：故障转移、能力检查、按延迟/价格选模型、逐请求成本统计。全程本地。

AgentGate is not a hosted API reseller. It is a local-first desktop workbench for people who use coding agents and want provider flexibility — without breaking client-specific behavior.

不是托管 API 分发平台，是给编程 agent 用户的本地桌面工作台。要 provider 自由，又不愿客户端特定能力被破坏。
