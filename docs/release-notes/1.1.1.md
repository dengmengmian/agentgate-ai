## AgentGate 1.1.1 — More themes and a sharper icon / 更多主题，更清晰的图标

### What's new
- **Eight themes to choose from.** Warm Amber, Sunny Day, Steel Blue, Pine Forest, Purple Night, Rice Beige, Misty Blue, and Sakura Pink — the settings page now shows a 2×4 swatch preview.
- **Redrawn app icon.** A beige rounded-square base with a tri-color atomic orbit, so the shape is now legible in the macOS dark dock.
- **Fixed the `agentgate-serve` CLI build.** The binary failed to compile before.

### 本次更新
- **主题扩展到 8 套。** 暖琥珀、晴日、钢蓝、松林、紫夜、米麻、雾蓝、樱粉。设置页换成 2×4 色板预览。
- **应用图标重画。** 米黄圆角矩形底 + 三色原子轨道，macOS 暗色 dock 里能看出形状了。
- **修复 `agentgate-serve` CLI 二进制编译失败。**

### Why it matters / 为什么重要
Personalization and visual polish, plus a fix that keeps the CLI build healthy. 更多个性化选择和视觉细节，同时修好了 CLI 构建。
