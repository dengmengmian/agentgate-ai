## AgentGate 1.2.3 — Stable defaults and visible degradations / 更稳的默认配置，可见的能力降级

### What's new
- **DeepSeek defaults move to v4.** Auto-setup no longer targets the soon-to-be-retired `deepseek-chat` / `deepseek-reasoner` models.
- **MiMo Token Plan stays in one region.** `sk-*` keys use the open API domain; `tp-*` keys use `token-plan-{cn|sgp|ams}` and keep a consistent regional host.
- **MiMo `web_search` degrades automatically.** Token Plan strips it up front; pay-as-you-go strips and retries once when the plugin isn't enabled.
- **Claude Code direct connections drop the `[1m]` suffix.** Recommended mappings now use plain model IDs.
- **Capability degradations are now diagnosable.** Image stripping, `web_search` downgrades, MCP advisories and more are written to each request log's `degradation_events`.

### 本次更新
- **DeepSeek 默认模型收敛到 v4。** 自动配置不再选用即将下线的 `deepseek-chat` / `deepseek-reasoner`。
- **MiMo Token Plan 区域域名保持一致。** `sk-*` 用开放 API 域名，`tp-*` 用 `token-plan-{cn|sgp|ams}` 并保持同一区域 host。
- **MiMo `web_search` 自动降级。** Token Plan 预剥离；按量付费遇 Plugin 未开通时剥离后重试一次。
- **Claude Code 直连模型不再写 `[1m]` 后缀。** 默认推荐映射用普通模型 ID。
- **能力降级可诊断。** 图片剥离、`web_search` 降级、MCP advisory 等事件写入请求日志的 `degradation_events`。

### Why it matters / 为什么重要
Defaults stay aligned with upstream lifecycle changes, and every silent fallback now leaves a trail. 默认配置跟上游下线节奏对齐，每一次降级都在请求日志里留痕，排查不再靠猜。
