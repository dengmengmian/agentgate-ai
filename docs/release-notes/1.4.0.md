## AgentGate 1.4.0

### English Summary

- Added GitHub Copilot provider support for Claude Code and Codex, including token exchange, credential renewal, model-name normalization, and `x-initiator` request classification.
- Added Windows coverage for running-client detection, Codex restart after applying config, and Claude Desktop integration.
- Improved Claude thinking behavior, Anthropic prompt-cache injection, fastest-route cold starts, circuit-breaker tuning, and actionable errors for polluted sessions.
- Fixed dashboard performance on large request-log databases, streaming UTF-8 corruption, Windows HOME fallback issues, Claude Code compaction detection, Kimi reasoning parameter forwarding, and Chinese long-session compaction estimates.
- Expanded redaction for request logs and diagnostic bundles, and added Host / Origin checks for headless mode.

### 中文摘要

- 新增 GitHub Copilot Provider：可用 Copilot 订阅跑 Claude Code / Codex，包含 token 交换、凭证续期、模型名归一化和 `x-initiator` 计费分类。
- 补齐 Windows 客户端集成：运行中客户端检测、应用配置后自动重启 Codex、Claude Desktop 集成。
- 改进 Claude thinking、Anthropic prompt cache 注入、最快路由冷启动、熔断阈值和会话污染错误提示。
- 修复大库首屏慢、流式中文/emoji 乱码、Windows HOME 缺失、Claude Code 压缩误判、Kimi 参数透传和中文长对话压缩估算。
- 扩展请求日志 / 诊断包脱敏，并为 headless 模式增加 Host / Origin 校验。
