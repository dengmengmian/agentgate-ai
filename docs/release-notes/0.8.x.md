## AgentGate 0.8.x — Smoother installs and releases / 安装与发版更顺滑

### What's new
- **Cleaner first run and release pipeline.** A fresh install with an empty log table no longer throws a database error, off-screen pets are pulled back into view, and the desktop package no longer bundles the headless CLI (now shipped as a separate artifact). The macOS / Docker release flow is fixed too.

### 本次更新
- **首次启动和发版更顺。** 空日志表首次安装不再报数据库错误，跑到屏幕外的桌宠会被拉回可见区域，桌面包不再夹带 headless CLI（改为独立产物）。macOS / Docker 发版流程一并修复。

### Why it matters / 为什么重要
A batch of install and release fixes that smooth out the rough edges around getting AgentGate up and shipped. 一批安装与发版修复，磨平了上手和分发的毛刺。
