## AgentGate 1.3.4 — Your desktop pet learns to talk / 桌宠会聊天了

### What's new
- **Chat with your desktop pet.** Double-click the pet to open a chat box. Nine characters each have their own tone and reactions, remember things you tell them (like your name) across restarts, and show the real reason when a call fails — no main provider set, missing API key, or call error — instead of an unrelated greeting.
- **Native right-click menu, click-through, and pokes.** A native context menu switches characters, opens gateway / logs / settings, clears memory, or hides the pet without the window blocking what's underneath. A click-through mode lets clicks pass to the app below, and a single click triggers a character-specific reaction.
- **Smaller window and big performance gains.** The default window shrank to 140×200 and grows only when a bubble or chat appears. Polling and rendering were heavily optimized so the pet is near-zero cost in the background.

### 本次更新
- **和桌宠聊天。** 双击宠物开聊天框，9 个角色各有专属语气和反应，记住你说过的事（如名字）跨重启保留；失败时显示真实原因（未设主供应商 / API key 缺失 / 调用错误）而不是无关问候。
- **原生右键菜单、鼠标穿透、单击戳一下。** 原生右键菜单可切换角色、打开网关 / 日志 / 设置、清空记忆、隐藏宠物，不再因窗口展开挡住下方应用；鼠标穿透模式让点击穿到下方应用；单击触发角色专属反应。
- **窗口更小、性能大幅优化。** 默认窗口缩到 140×200，气泡或聊天框出现时才按需撑高；轮询与渲染深度优化，后台几乎零消耗。

### Why it matters / 为什么重要
This is the release where the desktop pet becomes a genuine, sticky feature: it talks, remembers, stays out of the way, and barely costs anything in the background — turning a mascot into a daily companion. 这一版让桌宠从装饰变成真正有黏性的功能：会聊、能记、不挡屏、后台几乎零开销，把吉祥物变成日常陪伴。
