## AgentGate 0.2.x — Smoother edits and quieter updates / 编辑更顺手，更新更省心

### What's new
- **Friendlier model mapping.** The dropdown now lets you pick from a list or type your own value, and it stays readable in dark mode.
- **Signed builds with auto-update on macOS.** AgentGate ships ad-hoc signed and updates itself in the background via `.app.tar.gz`.

### 本次更新
- **模型映射更好用。** 下拉框既能从列表里选，也能手动输入，深色主题下依然清晰可读。
- **macOS 签名 + 自动更新。** 应用走 ad-hoc 签名，并通过 `.app.tar.gz` 在后台自动更新。

### Why it matters / 为什么重要
A small interim batch focused on day-to-day usability and keeping the app current without manual reinstalls. 一个过渡性的小合集，专注日常易用性，让应用无需手动重装即可保持最新。
