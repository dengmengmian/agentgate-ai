## AgentGate 0.1.0 — The first public release / 首个公开版本

### What's new
- **A protocol-converting gateway** that translates between OpenAI Responses, Anthropic Messages, and Chat Completions.
- **Multi-provider management** for DeepSeek, OpenAI, OpenRouter, Kimi, and custom providers.
- **Route Profiles** with multi-provider priority and failover.
- **One-click tool setup** for Codex, Claude Code, and OpenCode.
- **Usage and cost tracking** with model mapping, custom headers, reasoning effort passthrough, and an auto-refreshing dashboard.
- **Request logs, self-diagnostics, and an exportable diagnostic bundle.**
- **System tray, launch-at-startup, bilingual UI, and automatic config backups.**

### 本次更新
- **协议转换网关**，在 OpenAI Responses / Anthropic Messages / Chat Completions 之间互转。
- **多 provider 管理**，支持 DeepSeek / OpenAI / OpenRouter / Kimi / 自定义。
- **Route Profile 路由配置**，多 provider 优先级 + failover。
- **工具一键配置**，支持 Codex / Claude Code / OpenCode。
- **用量与费用统计**，含模型映射、自定义请求头、reasoning effort 透传、自动刷新的 Dashboard。
- **请求日志、诊断自检、可导出的诊断包。**
- **系统托盘、开机自启、中英双语、配置自动备份。**

### Why it matters / 为什么重要
This is where AgentGate begins: a personal tool open-sourced into a public release, already a complete local AI gateway on day one. AgentGate 的起点：一个自用工具开源成公开版本，从第一天起就是一个完整的本地 AI 网关。
