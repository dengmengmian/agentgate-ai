## AgentGate 1.4.3

### English Summary

- Added Claude Code status alerts for the desktop pet through Claude Code hooks and a local file mailbox.
- Added system notifications when Claude Code is waiting for permission or has finished the current turn; working events stay quiet to avoid notification spam.
- Improved pet bubble priority so Claude Code waiting/done messages are not immediately replaced by ordinary pet bubbles.
- Added GitHub Discussions links to the README navigation.

### 中文摘要

- 新增 Claude Code 桌宠状态提醒：通过 Claude Code hooks 和本地文件信箱接收状态事件。
- 新增系统通知：Claude Code 等待授权或本轮完成时通知用户；工作中事件不发系统通知，避免刷屏。
- 优化桌宠气泡优先级：CC 等待/完成提示不会被普通气泡立即顶掉。
- README 顶部新增 GitHub Discussions / 社区讨论入口。
