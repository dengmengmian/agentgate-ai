## AgentGate 0.1.2 — Backups, restore, and small fixes / 备份、恢复与小修复

### What's new
- **Backup history with one-click restore** on the tools page.
- **Accurate version number**, now read from the Tauri API at runtime instead of being hardcoded.
- **Quieter update checks** — failures are handled silently.
- **Codex default model** updated to `gpt-5.5`.

### 本次更新
- **备份历史 + 一键恢复**，在工具页。
- **版本号准确**，从 Tauri API 运行时读取，不再硬编码。
- **检查更新更安静** —— 失败时静默处理。
- **Codex 默认模型** 更新到 `gpt-5.5`。

### Why it matters / 为什么重要
A small maintenance release that makes config changes safer to undo. 一个小维护版本，让配置改动更容易回退。
