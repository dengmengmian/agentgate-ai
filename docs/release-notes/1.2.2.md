## AgentGate 1.2.2 — Your pet looks right on Windows / Windows 上的桌宠不再是白卡片

### What's new
- **The desktop pet no longer shows a white background on Windows.** Windows' WebView2 control defaults to an opaque background, so the pet window now gets an explicit dark background and renders properly. macOS behavior is unchanged.

### 本次更新
- **Windows 桌面宠物白色背景修复。** Windows 的 WebView2 控件默认底色不透明，给 pet 窗口加了显式深色背景，宠物在 Windows 上不再是个白卡片。macOS 行为不变。

### Why it matters / 为什么重要
A small but visible polish for Windows users. Windows 用户的桌宠终于显示正常。
