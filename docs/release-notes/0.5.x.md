## AgentGate 0.5.x — Standalone CLI and startup cleanup / 独立 CLI 与启动清理

### What's new
- **The standalone `agentgate-serve` CLI now ships with every release.** Plus a cleaner startup: no more example request logs (`req-seed-*`), correct per-platform CLI build outputs, and the tray menu picks up Chinese correctly when launched from the macOS Dock/Finder.

### 本次更新
- **独立 `agentgate-serve` CLI 开始随版本一起发。** 启动也更干净：不再有示例请求日志（`req-seed-*`），各平台 CLI 构建输出隔离，从 macOS Dock/Finder 启动时托盘菜单正确识别中文。

### Why it matters / 为什么重要
Small but useful: the headless CLI becomes a real release artifact, and the app starts clean. 小而实用：headless CLI 成为正式发布产物，应用启动更干净。
