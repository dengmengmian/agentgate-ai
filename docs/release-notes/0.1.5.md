## AgentGate 0.1.5 — No more crashes on Chinese text / 中文内容不再让网关崩溃

### What's new
- **Chinese (and other multibyte) text no longer crashes the gateway.** All string truncation now cuts on safe character boundaries.
- **Codex stays alive on big tool output.** The 4000-byte truncation limit is gone; tool output is passed through as-is.
- **Clearer errors and safer keys.** Error responses carry a `type` field instead of showing "Unknown error", auth failures return a proper 401, and API keys display as a fixed mask like `sk-1****cdef`.
- **A faster, tidier backend.** Stats now load from 3 queries instead of 14, request logs are indexed by timestamp, and deleting a provider cleans up its routes.

### 本次更新
- **中文等多字节内容不再让网关崩溃。** 所有字符串截断改用安全字符边界。
- **大 tool output 不再拖垮 Codex。** 移除 4000 字节截断限制，tool output 原样透传。
- **错误更清楚、密钥更安全。** 错误响应带 `type` 字段，不再显示「Unknown error」；认证失败正确返回 401；API Key 显示为固定遮罩 `sk-1****cdef`。
- **后端更快更干净。** 统计查询从 14 条 SQL 合并为 3 条，请求日志按时间戳建索引，删除 provider 时同步清理路由。

### Why it matters / 为什么重要
A stability-first release that removes the crashes and data leaks most likely to bite real Chinese-language users. 一个以稳定性为先的版本，重点解决中文用户最容易踩到的崩溃和密钥泄露问题。
