## AgentGate 0.5.0 — Native Gemini and headless serving / 原生 Gemini 与无头服务

### What's new
- **Native Gemini format support.** Two-way Codex ↔ Gemini conversion means Gemini CLI can reach any provider through AgentGate, including DeepSeek and Kimi.
- **Task-level smart routing.** AgentGate auto-picks the provider and model based on input length, images, tools, and system keywords.
- **Headless serving mode.** Run AgentGate as the `agentgate-serve` CLI binary or via Docker, with dates shown in logs (MM-DD HH:MM:SS).

### 本次更新
- **Gemini 原生格式支持。** Codex ↔ Gemini 双向转换，Gemini CLI 可通过 AgentGate 接 DeepSeek / Kimi 等任意 provider。
- **任务级智能路由。** 按输入长度 / 图片 / 工具 / 系统关键词自动选 provider 和模型。
- **无头服务模式。** 通过 `agentgate-serve` CLI 二进制或 Docker 运行，日志显示日期（MM-DD HH:MM:SS）。

### Why it matters / 为什么重要
AgentGate stops being just a desktop app: native Gemini support and a headless serving mode let it run as smart routing infrastructure for any client, anywhere. AgentGate 不再只是桌面应用：原生 Gemini 支持和无头服务模式让它能作为智能路由基础设施，为任意客户端、任意环境服务。
