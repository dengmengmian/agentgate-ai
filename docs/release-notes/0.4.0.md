## AgentGate 0.4.0 — Know your costs and stay online / 看清成本，保持在线

### What's new
- **Cost tracking out of the box.** Built-in pricing for 22 models computes the cost of every request and surfaces total / today / average on the dashboard; the pricing table is editable inline or overridable in settings.
- **Multi-account rotation.** One provider can hold multiple API keys (a JSON array) and auto round-robins between them.
- **Smarter prompt caching.** The Codex → Anthropic path auto-injects `cache_control` on system / tools / last assistant, and cache capability is auto-detected on connection test.
- **Provider health at a glance.** Each card embeds 1h / 24h success rate, average and P95 latency, and request count.
- **Resilient retries.** 429 / 500 / 502 / 503 retry with backoff, respect `Retry-After`, and switch keys on each attempt.

### 本次更新
- **开箱即用的费用追踪。** 内置 22 个模型价格，每条请求自动算费，仪表盘展示总费用 / 今日 / 平均，设置页价格表可内联编辑或自定义覆盖。
- **多账号轮转。** 同 provider 支持多 API Key（JSON 数组），自动 round-robin。
- **更智能的 Prompt Cache。** Codex → Anthropic 转换路径自动给 system / tools / 最后 assistant 打 `cache_control`，测试连接时自动探测缓存能力。
- **Provider 健康一目了然。** 卡片内嵌 1h / 24h 成功率、平均延迟、P95 延迟、请求数。
- **更稳的重试。** 429 / 500 / 502 / 503 自动退避重试，尊重 `Retry-After`，每次重试换 Key。

### Why it matters / 为什么重要
0.4.0 turns AgentGate into a cost-aware, reliable gateway: you finally see what every request costs, and multi-key rotation plus smart retries keep you online when a single key throttles or fails. 0.4.0 让 AgentGate 成为成本可见、可靠的网关：每条请求花多少钱看得清，多 Key 轮转加智能重试，让单个 Key 限流或失败时也不掉线。
