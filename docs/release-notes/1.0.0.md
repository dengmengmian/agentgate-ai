## AgentGate 1.0.0 — Official release / 正式发布

### What's new
- **AgentGate is officially 1.0.** A stable, production-ready local AI gateway: unify every official AI API behind one local endpoint, manage multiple providers, smart routing, failover, cost tracking, request logs, and diagnostics.

### 本次更新
- **AgentGate 正式发布 1.0。** 稳定可用的本地 AI 网关：把各家 AI 官方接口统一到一个本地入口，支持多 provider 管理、智能路由、failover、成本统计、请求日志和诊断。

### Why it matters / 为什么重要
1.0.0 marks the milestone from preview to production: the core gateway is stable enough to be your everyday entry point for AI coding tools. 1.0.0 是从预览走向正式可用的里程碑：核心网关已足够稳定，可以作为日常 AI 编码工具的统一入口。
