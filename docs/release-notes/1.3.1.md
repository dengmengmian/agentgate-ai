## AgentGate 1.3.1 — Edit your agent instructions, with safety nets / 直接管你的 Agent 指令，还能一键回滚

### What's new
- **Edit your global instructions inside AgentGate.** Work on `~/.claude/CLAUDE.md` / `~/.codex/AGENTS.md` directly, with 4 built-in templates (minimal Chinese rules / TDD / code review / security audit). Every save snapshots first, so you can roll back in one click.
- **Client config history and one-click rollback.** Each "apply / disable / switch" snapshots the on-disk config (initial version + last 10), and a new "history" button lets you restore after confirmation.
- **Measure node speed.** The Providers page fires a 1-token probe at every enabled provider in parallel, sorts by latency, and color-codes connect / TTFB / total.
- **Gateway refinement layer.** Three opt-in switches under Settings → General: request field filtering, reasoning-param correction (`thinking.budget_tokens` / `reasoning.effort`), and error normalization (`context_length_exceeded`). Ships with DeepSeek / MiMo / Anthropic / OpenAI / Kimi rules.

### 本次更新
- **在 AgentGate 内编辑全局指令。** 直接改 `~/.claude/CLAUDE.md` / `~/.codex/AGENTS.md`，4 个内置模板（极简中文规范 / TDD / 代码评审 / 安全审计）可覆盖或追加。写盘前自动 snapshot，可一键回滚。
- **客户端配置版本史与一键回滚。** 每次「应用 / 关闭 / 切换」前自动 snapshot 盘上配置（保留初始版 + 最近 10 条），卡片新增「历史」按钮，确认后一键回滚。
- **节点测速。** Providers 页对所有启用 provider 并行发 1-token 探测，按延迟排序，连接 / TTFB / 总耗时三段染色。
- **网关精炼层。** 「设置 → 通用」三项默认关的开关：请求字段过滤、推理参数校正（`thinking.budget_tokens` / `reasoning.effort`）、错误响应归一（`context_length_exceeded`）。内置 DeepSeek / MiMo / Anthropic / OpenAI / Kimi 规则。

### Why it matters / 为什么重要
AgentGate stops being just a gateway and starts managing the **whole agent setup** — instructions, client configs, provider health — with rollback as a first-class safety net. AgentGate 从单纯网关走向管理 **整套 agent 配置**——指令、客户端配置、供应商健康——并把一键回滚做成默认的安全网。
