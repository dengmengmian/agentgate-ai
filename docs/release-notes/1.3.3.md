## AgentGate 1.3.3 — See the full request story, top to bottom / 一眼看清每个请求的完整链路

### What's new
- **One view, the whole request.** Open any request and see route, provider, model, status, tokens, cost, fallback and the error trail in a single place — no more piecing it together across screens.
- **Conversations and instructions now render real Markdown.** Tables, task lists and code blocks show up properly in session transcripts and global-instruction previews.

### 本次更新
- **一个视图看完整请求。** 打开任意请求，路由、供应商、模型、状态、tokens、成本、fallback、错误链路摘要全在同一处，不用再来回拼。
- **对话和指令支持真正的 Markdown。** 会话记录和全局指令预览接入 GFM 渲染，表格、任务列表、代码块都能正常显示。

### Why it matters / 为什么重要
AgentGate keeps turning raw gateway traffic into something you can actually read. This release makes a single request **self-explanatory** and cleans up the logs page so problems jump out faster. 这一版把单个请求做成 **自解释**，并让日志页更易扫，问题更快浮现。
