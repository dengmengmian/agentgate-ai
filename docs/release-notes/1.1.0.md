## AgentGate 1.1.0 — Codex.app, MiMo, and cache hit-rate you can see / Codex.app、MiMo 接入，缓存命中率看得见

### What's new
- **Codex.app IDE plugin now just works.** Hijacking the OpenAI provider with `requires_openai_auth = true` lets the IDE plugin, Browser, Mobile, and quota lookups all run through AgentGate while your ChatGPT OAuth tokens stay untouched. Compressed request bodies (Codex.app zstd, ChatGPT desktop gzip) no longer 500.
- **Xiaomi MiMo as a first-class provider.** All 5 chat models, multi-turn reasoning_content, built-in pricing, and Codex `web_search_preview` translated through to MiMo `web_search`.
- **Higher prompt cache hits.** Session affinity binds a session to its provider for 1 hour when the upstream returns `cached_tokens > 0`, so follow-up requests reuse the same provider. The Dashboard now shows a cache footer with an auto-computed hit rate.
- **Seamless failover on bad first frames.** When an upstream returns HTTP 200 but stuffs an error (quota / ban / rate-limit) into the first SSE event, AgentGate switches to the next provider with zero client impact.
- **Rebuilt Dashboard with live KPIs.** Today's 5 core metrics plus a 7-day bar chart, a time-range switcher (today / 7 / 14 / 30 days), and a 6-item live footer refreshing every 5 seconds.
- **Per-model capability matrix.** Toggle 8 capabilities (text / vision / audio_in / tts / video_in / reasoning / tools / web_search) per model; image requests auto-swap to a vision-capable model, and unchecking web_search stops it being sent.
- **Sharper logs.** Pagination at 100 rows per page and precise client identification by User-Agent (Codex / Claude Code / OpenCode / Cursor / Cherry Studio / Continue / Cline / Roo Code).
- **Tray menu live status.** Active provider, today's request count, gateway port, and a one-click "switch provider" submenu.

### 本次更新
- **Codex.app IDE 插件开箱即用。** 改用「劫持 OpenAI provider + `requires_openai_auth = true`」，IDE 插件 / Browser / Mobile / 配额查询全部走 AgentGate，ChatGPT OAuth tokens 完全保留。Codex.app 的 zstd、ChatGPT 桌面客户端的 gzip 压缩请求体不再 500。
- **小米 MiMo 一等公民。** 完整 5 个聊天模型、多轮 reasoning_content、内置定价，Codex 的 `web_search_preview` 翻译为 MiMo `web_search`。
- **缓存命中率更高。** 上游回 `cached_tokens > 0` 时，会话与 provider 绑定 1 小时，后续请求复用同 provider。Dashboard 新增缓存页脚，命中率自动计算。
- **首段错误也能无感切换。** 上游 HTTP 200 但首段 SSE 塞错误（quota / ban / rate-limit）时自动切下个 provider，客户端零感知。
- **Dashboard 重做，KPI 实时。** 今日 5 个核心指标 + 7 天柱状图，今天 / 7 / 14 / 30 天切换，6 项指标页脚每 5 秒刷新。
- **每模型能力矩阵。** 8 种能力（text / vision / audio_in / tts / video_in / reasoning / tools / web_search）逐模型勾选；带图请求自动 swap 到支持 vision 的模型，取消勾选 web_search 即停止下发。
- **日志更清晰。** 每页 100 条分页，按 User-Agent 精确识别客户端（Codex / Claude Code / OpenCode / Cursor / Cherry Studio / Continue / Cline / Roo Code）。
- **托盘菜单实时状态。** 当前 active provider、今日请求数、网关端口，「切换服务商」子菜单一键切换。

### Why it matters / 为什么重要
This is the release that makes AgentGate a serious daily driver for IDE coding agents: full Codex.app compatibility, MiMo as a native provider, and cache mechanics (session affinity + visible hit rate) that turn prompt caching from a black box into a number you can optimize. 这一版让 AgentGate 真正成为 IDE 编码助手的日常入口：完整兼容 Codex.app、原生接入 MiMo，并把 prompt cache 从黑盒变成可优化的命中率指标（会话亲和 + 可视命中率）。
