## AgentGate 0.1.4 — One-click setup for more coding tools / 更多编码工具一键接入

### What's new
- **One-click switch for Claude Code.** Same save / restore mechanism already used for Codex.
- **One-click setup for OpenCode.** AgentGate writes `~/.config/opencode/opencode.json` for you.
- **A real routing system.** Three default routes are created per protocol, new providers join every route chain automatically, and the UI lets you create routes and rename them inline.
- **A clearer navigation layout.** Overview → Providers → Routes → Gateway → Clients → Logs → Diagnostics → Settings.

### 本次更新
- **Claude Code 一键切换。** 与 Codex 相同的 save / restore 机制。
- **OpenCode 一键配置。** AgentGate 帮你写好 `~/.config/opencode/opencode.json`。
- **完整的路由系统。** 按协议自动建 3 个默认路由，新增 provider 自动加入所有路由链，UI 支持创建路由和 inline 重命名。
- **更清晰的导航。** 概览 → 服务商 → 路由 → 网关 → 客户端 → 日志 → 诊断 → 设置。

### Why it matters / 为什么重要
AgentGate grows from a Codex helper into a multi-client gateway with a proper routing model. AgentGate 从 Codex 助手成长为带有完整路由模型的多客户端网关。
