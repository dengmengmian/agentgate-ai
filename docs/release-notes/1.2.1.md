## AgentGate 1.2.1 — Quick setup that just works / 快速配置一次到位

### What's new
- **MiMo / DeepSeek no longer auto-configure `[1m]` long-context models.** Plain model IDs are enough and avoid upstream `Not supported model` errors.
- **Quick setup tests the right provider.** A newly created provider is set active immediately, so connection tests no longer run against the old route.
- **Quick setup fills in models and detects capabilities.** Same flow as "Add provider → fetch and detect capabilities".
- **Failed connection tests show the real reason.** No more vague errors.

### 本次更新
- **MiMo / DeepSeek 不再自动配置 `[1m]` 长上下文模型。** 普通模型 ID 已经够用，避免上游返回 `Not supported model`。
- **快速配置测试连接打到正确 provider。** 创建后立即设为 active，不再沿用旧路由测试。
- **快速配置补齐模型 + 能力识别。** 与「添加供应商 → 拉取并识别能力」流程一致。
- **连接测试失败显示具体原因。** 不再笼统报错。

### Why it matters / 为什么重要
Quick setup is now consistent with the full add-provider flow and fails loudly with actionable errors. 快速配置体验与完整添加流程一致，出错时直接告诉你原因。
