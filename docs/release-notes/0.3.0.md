## AgentGate 0.3.0 — One gateway, 23 providers / 一个网关，接入 23 家服务商

### What's new
- **23 built-in provider presets, up from 7.** Add Google Gemini, xAI, Mistral, Groq, Together, Fireworks, Cerebras, Perplexity, Cohere, Zhipu GLM, Qwen, SiliconFlow, Volcengine, Baichuan, StepFun, 01.AI and more without hunting for endpoints.
- **One-click setup for Gemini CLI and AtomCode.** AgentGate writes their config files for you and lets you flip between the official service and AgentGate.
- **A steadier streaming experience.** Fewer stalls and hangs across Anthropic SSE streams and vision routing.

### 本次更新
- **内置 provider 预设从 7 个扩充到 23 个。** 新增 Google Gemini、xAI、Mistral、Groq、Together、Fireworks、Cerebras、Perplexity、Cohere、智谱 GLM、通义千问、硅基流动、火山引擎、百川、阶跃星辰、零一万物等，不用再自己找接入地址。
- **Gemini CLI 和 AtomCode 一键配置。** AgentGate 帮你写好配置文件，并支持在官方和 AgentGate 之间切换。
- **流式更稳。** Anthropic SSE 流和识图路由的卡顿、挂起问题大幅减少。

### Why it matters / 为什么重要
This release turns AgentGate into a true multi-provider hub: most mainstream model vendors are now one click away. 这一版让 AgentGate 真正成为多服务商中枢，主流模型厂商基本都能一键接入。
