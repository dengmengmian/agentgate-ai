## AgentGate 1.3.2 — Claude Desktop in, smarter routing, accurate cost / 接入 Claude Desktop，路由更聪明，成本算得准

### What's new
- **Point Claude Desktop at AgentGate (macOS).** One click to route Claude Desktop's third-party inference gateway through AgentGate, with history and rollback.
- **Read the whole conversation from the logs.** The "session" view opens the full Claude Code / Codex chat, plus a one-click copy of the resume command.
- **Routing that picks for you.** On failover, choose "cheapest" or "fastest" and AgentGate auto-selects a provider by price or latency.
- **Optional active health probing.** A background check pings providers on a schedule and shows the result on each card, without affecting actual routing (off by default).

### 本次更新
- **把 Claude Desktop 指向 AgentGate（macOS）。** 一键把 Claude Desktop 的第三方推理网关指到 AgentGate，可历史回滚。
- **从日志里读完整对话。** 「会话」视图点开就能看 Claude Code / Codex 的完整聊天记录，还附一键复制的恢复命令。
- **路由帮你挑。** 失败转移可选「最便宜」/「最快」，自动按单价或延迟挑供应商。
- **可选的主动健康探测。** 后台定期探活供应商，结果显示在卡片上，不影响实际路由（默认关）。

### Why it matters / 为什么重要
This is the release where AgentGate becomes a **real multi-client hub**: more clients can route through it, routing gets cost- and latency-aware, and cost accounting finally gets accurate. 这一版让 AgentGate 真正成为 **多客户端枢纽**：接入更多客户端、路由开始考虑成本与延迟、成本统计终于算得准。
